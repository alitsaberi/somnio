# Getting started

## Installation

ZUtils requires **Python 3.10+**.

### With uv (recommended)

```bash
uv add zutils
```

To use the [CLI](./user-guide/cli.md), install with the optional `cli` extra:

```bash
uv add zutils[cli]
```

### With pip

```bash
pip install zutils
```

For CLI support:

```bash
pip install zutils[cli]
```
